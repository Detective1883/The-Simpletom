On the Subject of The Simpletom

'Okay! Let's take attendance!'

The Simpletom looks a lot like The Simpleton. The one difference is the text on the button. Depending on what it says, you will need to input a different code. That's on you to figure out.

Appendix Attendance

LabelInstructions
"The Simpletom"Mash the button X times, where X = (4*(indicators) - 3), modulo 4*
"The Simplebob"Hold the button for 3 real-life seconds, then release
"The Simplejack" or "The Simplejill"Submit "HILL" in Morse Code (see Morse Code)
"The Simpletim"Submit "TIM" in Tap Code (see Appendix Poke)
